Программа gen_lk_4_31_31 принимает один аргумент командной строки — 
имя семейства ЛК с симметрией (4,31,31), и записывает все существенно 
различные ЛК этого семейства в выходной файл 
family_4_31_31_<имя_семейства>.txt.

Для массового построения семейств используте скрипт gen_4_31_31.bat, 
который читает имена семейтв из файла names.txt.

Syntax:

```
gen_lk_4_31_31 FAMILY
FAMILY: family name of Latin Squares with (4,31,31) symmetry
```

The program will generate all Latin Squares in the family and write them to file
family_4_31_31_FAMILY.txt
