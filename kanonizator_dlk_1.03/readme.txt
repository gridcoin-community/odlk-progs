Программа канонизации ДЛК10.

Для работы программы нужен файл hash_tabl.bin, поместите его в одну папку
с программой.

По умолчанию имя входного файла — input.txt, а выходного — output.txt. При
помощи аргументов командной строки это можно изменить, например:
kanonizator_dlk.exe input_my_dlk.txt
kanonizator_dlk.exe input_my_dlk.txt out_my_dlk.txt

Так же предусмотрен один ключ — /m для подавления запроса на выход из
программы (/m значит mute).

КФ выводятся в отсортированном порядке без повторов.
